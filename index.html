Pause — you scanned an unknown QR code.

QR codes can hide links. The risky part is that you often don’t see the real destination until it’s too late.

Safety check

  1. Preview the URL before opening (most scanners show it).
  2. Read the domain carefully — scammers use look-alike spelling.
  3. Avoid shortened links (bit.ly/tinyurl) unless you trust the source.
  4. If it asks for downloads or payment right away → close it.
  5. When unsure, type the official site yourself instead of using the QR.

Tip: HTTPS helps, but it doesn’t guarantee the page is legitimate.

What could have happened?

Fake portal link

Tries to trick you into entering credentials (phishing).

Payment trap

“Fine due”, “verification fee”, “parking ticket”.

Malware / app prompt

Pushes an APK/app install or “security update”.

Subscription scam

Hidden recurring charges via fake forms.

Safe scanning habits

Verify the source

If it’s on a poster, check the official account/site of the organizer.

Type official URLs

For portals/payments, type the site yourself instead of trusting the QR.

Be suspicious of urgency

“Act now”, “account suspended”, “fine due today” are classic scam triggers.

Don’t install apps

If scanning leads to an install prompt, stop and close it.
